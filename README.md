# 码上点（扫码点餐系统）

这是一个扫码点餐系统，旨在通过手机点餐的方式，达到方便商家和客户的双赢。

---

**用户分客户和商家：**

客户扫码，进入指定商家的微信点餐界面进行点餐操作。

商家则通过PC端登入，进行查看订单，修改菜单等操作。

**优势：**

商家：减少商家的人力资源成本，提高客户点餐效率。

客户：更自由方便的点餐环境，更丰富的菜品介绍。
